# Moonrepo and proto Plugin

Moonrepo (タスクランナー) と proto (ツールチェーン管理) の統合的な利用をサポートするプラグインです。

## 特徴

- **ツールチェーン管理**: `proto` を使用した一貫性のある言語バージョンの管理方法。
- **タスク構成**: `moon.yml` での効率的なタスク定義とキャッシングの活用。
- **ワークスペース管理**: モノレポ構成と CI/CD 連携のベストプラクティス。

## 構成

- `skills/moon-proto/SKILL.md`: メインエントリポイント
- `skills/moon-proto/guides/`: 詳細ガイドディレクトリ
    - `proto-management.md`: ツールチェーン管理と `.prototools`
    - `moon-tasks.md`: タスク定義と実行
    - `workspace-config.md`: ワークスペース設定と CI 連携
