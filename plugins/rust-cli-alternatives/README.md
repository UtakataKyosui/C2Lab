# rust-cli-alternatives

Modern Rust-based CLI tools that replace and enhance classic Unix commands.

## Overview

This plugin provides a comprehensive knowledge base for 30+ Rust-reimplemented CLI tools, covering installation, configuration, usage examples, and migration guides from classic Unix commands.

## Tools Covered

### File Management & Navigation
- **eza** — `ls` replacement with Git integration and icons
- **lsd** — `ls` replacement with Nerd Font icons
- **fd** — `find` replacement with intuitive syntax
- **broot** — `tree` replacement with fuzzy search and navigation
- **tre-command** — `tree` replacement with numbered entries
- **zoxide** — `cd` replacement with frecency-based directory jumping

### Text Processing & Search
- **ripgrep (rg)** — `grep` replacement, fastest code search tool
- **bat** — `cat` replacement with syntax highlighting and Git diffs
- **sd** — `sed` replacement with intuitive regex syntax
- **grex** — regex generator from example strings
- **hexyl** — `hexdump` replacement with colorized output

### System Monitoring
- **bottom (btm)** — `top`/`htop` replacement with TUI widgets
- **procs** — `ps` replacement with colorized output and tree view
- **dust** — `du` replacement with visual bar charts
- **bandwhich** — network monitoring per process

### Git & Version Control
- **delta** — `diff` pager with syntax highlighting
- **gitui** — Git TUI for keyboard-driven operations

### Development Tools
- **tokei** — line-of-code counter (replaces cloc)
- **hyperfine** — statistical benchmarking (replaces `time`)
- **just** — `make` replacement with simpler syntax
- **watchexec** — file watcher (replaces `watch`/`entr`)

### Networking
- **xh** — `curl`/httpie replacement with better UX
- **miniserve** — `python -m http.server` replacement

### Shell & Prompt
- **starship** — cross-shell prompt with smart detection
- **nushell** — structured data shell
- **tealdeer** — `tldr`/`man` replacement with examples
- **atuin** — shell history with sync and fuzzy search
- **yazi** — terminal file manager
- **jaq** — `jq` replacement written in Rust
- **ouch** — universal compression/decompression

## Skills

- **rust-cli-tools** — Comprehensive guide with installation, configuration, and usage examples
- **rust-cli-migration** — Quick cheatsheet for migrating from classic Unix commands

## Agent

- **rust-cli-tool-recommender** — Suggests Rust CLI alternatives when classic commands are used
