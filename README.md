# C2Lab

Claude Code（C2）の実験場（Lab）です。
Claude Codeでいちいち

- Hook
- Slash Command
- SubAgent
- Skill

を用意するのは面倒くさかったので、プラグインリポジトリを作成しました。

## ディレクトリ構成

- `plugins/`: Claude Code 用の各種プラグイン（ Hook, Slash Command, SubAgent, Skill など）
- `jj-config/`: Jujutsu (jj) の設定ファイル
- `.claude-plugin/`: プラグイン提供用メタデータ（marketplace.json）
